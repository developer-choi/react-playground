# 소개
<h3>Next, React를 자유롭게 공부하기위한 프로젝트 입니다.</h3>
1. 특정 라이브러리 또는 프레임워크를 공부하기 위한 예제페이지가 있습니다.
2. 특정 문제를 해결하기 위한 컴포넌트와 그 예제페이지가 있습니다.

# 실행방법
1. 반드시 [로컬 API 서버](https://github.com/developer-choi/express-playground)를 먼저 실행한 다음 이 프로젝트를 실행해야합니다.
2. Node.js 설치
3. git bash 설치
4. yarn install
5. yarn dev

# 주의사항
1. 개인프로젝트 이기 때문에, 제 주장 100%로 구현이 되어있으며, 실무에서는 이렇게 하면 안됩니다.<br/>
2. 남들이 하지 말라는 방법으로 작성된 코드와 주석, 커밋이 존재할 수 있습니다.<br/>
3. 이것은, 일부러 잘못된 방법으로 시도해보면서 정답을 찾아가는 과정이라고 봐주시면 됩니다.<br/>
4. 주석과 커밋메시지가 영어로 되어있는것 역시 단순히 영어에 익숙해지려고 한것입니다.<br/>
5. 불필요한 라이브러리들이 설치되어있을 수 있습니다. 이것또한 공부를 목적으로 하는것이며 실무에서는 프로젝트에 적합한것만 사용합니다.
