# 공부를 위한 프로젝트 입니다.
1. /pages/study: 공부를 위한 예제페이지가 있습니다.
2. /pages/solution: 문제를 해결한 예제페이지가 있습니다.
3. /pages/experimental: 문제를 해결중이거나 검증이 완료되지않은 예제페이지가 있습니다.

# 특이사항
1. 공부가 목적이기 때문에, 실무프로젝트와 차이점이 존재합니다.
2. "이 문제는 이렇게 해결하는것이 가장 좋은게 아닌가? (순전히 제 생각 대로)" 를 코드로 구현했기 때문입니다.
2. 실무에서는, 동료가 익숙한 방식과 스타일의 코드로 바꿔서 작성해야합니다.
3. 주석과 커밋메시지가 영어로 되어있다면, 단순히 영어에 익숙해지려고 작성했었습니다.
