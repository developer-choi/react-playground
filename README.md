# 공부를 위한 프로젝트 입니다.
1. /pages/study: 공부를 위한 예제페이지가 있습니다.
2. /pages/solution: 문제를 해결한 예제페이지가 있습니다.
3. /pages/experimental: 문제를 해결중이거나 검증이 완료되지않은 예제페이지가 있습니다.

# 특이사항
1. 개인프로젝트이면서, 목적이 공부이기때문에 실무프로젝트와 차이점이 존재합니다.
2. 실무에 적합하지 않은 코드와 주석이 존재합니다.
3. 커밋남겼던 단위도 실무에는 적합하지 않습니다.
3. 주석과 커밋메시지가 영어로 되어있는것은 단순히 영어에 익숙해지려고 한것입니다.
