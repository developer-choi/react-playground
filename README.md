### 공부를 위한 프로젝트 입니다.
1. /pages/solution: 문제를 해결한 예제페이지가 있으며, 대부분 실무에 실제로 적용해봤고, 의도대로 동작했습니다.
2. /pages/experimental: 문제를 "이런방향"으로 해결해봐야지 하는 텍스트를 코드로 표현하기만 한 예제페이지가 있고, 완성하지않았기에 실제로 동작하지는 않습니다.
3. /pages/study: 라이브러리/프레임워크를 공부/테스트한 예제페이지가 있습니다.

### 목표
1. 모든 React 개인프로젝트의 Best Practice를 예제형태로 모으고있습니다.
2. 프레임워크 / 라이브러리 설명중 이해가 안되는 부분을 이해하기위한 예제를 만들고있습니다.

### 특이사항1
1. 공부가 목적이기 때문에, 실무프로젝트와 차이점이 존재합니다.
2. "이 문제는 이렇게 해결하는것이 가장 좋은게 아닌가? (순전히 제 생각 대로)" 를 코드로 구현했기 때문입니다.
2. 회사에서는 회사의 컨벤션에 맞춰서 바꿔 작성합니다.
3. 주석과 커밋메시지가 영어로 되어있다면, 단순히 영어에 익숙해지려고 작성했었습니다.

### 특이사항2
1. 코드의 통일성이 없는 경우도 있습니다.
2. 페이지별로 예제를 모아놓은 성격이 강하기때문에,
3. 과거에 만든 페이지는 현재 저의 생각과 다를 수 있습니다. (= 최신화가 자주 되지않습니다.)
